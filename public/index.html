The Sopranos API

A RESTful API for quotes from the greatest TV show of all time (according to critics!)

Available Endpoints

1. Get All Quotes

Returns a list of all quotes. Optional filter by character.

GET /quotes

GET /quotes?character=tony

2. Get Random Quote

Returns a single random quote.

GET /quotes/random

3. Get All Quotes

Returns a quote by its ID.

GET /quotes/:id
